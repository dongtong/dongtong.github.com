Mac OS X Leopard 安装mysql

Jul 21st, 2012 | Comments

第二步: 让Mac OS X每次启动时启动MySQL服务

第三步: 查找MySQL的server socket

第四步: 配置server socket, 让PHP, RoR, Python,以及其他语言能够访问

第五步: 要想配置生效，先停下MySQL服务
